# Privacy Policy

Notion LaTeX Auto Paste does not collect, store, sell, transmit, or share user data.

## Data Handling

The extension processes clipboard text locally in the browser only when one of these actions happens:

- You paste text into Notion.
- ChatGPT writes text to the clipboard through its copy button.

The processed text is used immediately for the same paste or copy action. It is not saved.

## Network

The extension does not send network requests.

## Storage

The extension does not use Chrome storage, local storage, IndexedDB, cookies, or any external database.

## Permissions

The extension runs only on the sites declared in `manifest.json`:

- `https://app.notion.com/*`
- `https://notion.so/*`
- `https://*.notion.so/*`
- `https://chatgpt.com/*`

The Notion script fixes paste behavior. The ChatGPT script normalizes text when ChatGPT writes copied content to the clipboard.

## Contact

Open a GitHub issue in this repository for privacy or security questions.
