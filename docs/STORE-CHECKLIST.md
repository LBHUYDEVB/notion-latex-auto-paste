# Chrome Web Store Checklist

This project is prepared for GitHub distribution first.

Before submitting to Chrome Web Store, prepare:

- 128x128 extension icon
- 16x16, 32x32, and 48x48 icons if possible
- At least one 1280x800 screenshot
- Short description
- Detailed description
- Privacy policy URL
- Permission justification
- Single-purpose statement

Recommended permission explanation:

> This extension runs on Notion to normalize LaTeX paste content, and on ChatGPT to normalize copied answer text before it is pasted into Notion. It does not collect, store, or transmit user data.

For a lower-friction first store submission, consider publishing a Notion-only build and keeping the ChatGPT helper as an optional GitHub build.
